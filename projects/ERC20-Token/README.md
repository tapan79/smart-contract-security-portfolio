# ERC20 Token

## Overview

This project demonstrates an ERC20 token built using OpenZeppelin with additional custom functionality.

## Features

- ERC20 Standard
- ERC20Capped
- ERC20Burnable
- Initial Supply Mint
- Validator Reward (block.coinbase)
- Owner Access Control
- Configurable Reward
- Supply Cap Enforcement

## Smart Contract

- Token Name: token
- Symbol: TK

## Learning Objectives

- ERC20 Architecture
- Constructor Inheritance
- Function Overriding
- _update()
- _mint()
- Access Control
- OpenZeppelin Libraries
- Supply Cap
- Burnable Tokens
